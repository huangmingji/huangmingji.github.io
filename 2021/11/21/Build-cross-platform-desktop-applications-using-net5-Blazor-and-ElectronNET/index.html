使用.NET5、Blazor和Electron.NET构建跨平台桌面应用

发表于2021-11-21|更新于2023-02-13|ASP.NET Core
|阅读量:

Electron.NET是一个嵌入了ASP.NET Core的Electron的封装，通过Electron.NET可以构建基于.NET5的跨平台的桌面应用，使得开发人员只需要使用ASP.NET Core和 Blazor就可以胜任桌面应用的开发工作。
